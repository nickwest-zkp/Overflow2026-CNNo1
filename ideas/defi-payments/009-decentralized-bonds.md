# 去中心化债券平台 / Decentralized Bond Platform

> 发行链上债券，利息自动支付，到期自动赎回

## 解决什么问题

传统债券市场门槛高，中小企业和个人难以发行债券融资。债券发行、利息支付和到期赎回依赖多个中介机构，流程复杂且费用高昂。去中心化债券平台让任何实体都能在 Sui 上发行链上债券，利息通过 Move 合约自动支付，到期自动赎回本金，全程无需中介参与。

## 核心功能

- 债券创建引擎：自定义面值、票面利率、付息频率、到期日等参数
- 自动付息系统：每个付息日合约自动将利息转入持有人地址
- 到期自动赎回：到期日合约自动返还本金，无需持有人手动操作
- 债券二级市场：债券代币化后可在 DEX 上自由交易

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | BondContract 作为独立 Object，存储债券参数和状态 |
| PTB | 原子化执行"创建债券 → 锁定抵押 → 铸造债券代币" |
| Coin 标准 | 债券代币使用 Coin<T>，支持 DEX 交易 |
| 时间锁 | 基于时间戳控制付息日和到期日 |
| 动态字段 | 存储持有人列表、付息记录和赎回状态 |

## 实现方案

### 智能合约 (Move)
- `bond_factory` 模块：创建债券合约，定义参数和付息规则
- `coupon_payment` 模块：管理付息计划，在付息日自动分配利息
- `redemption` 模块：到期时处理本金赎回和合约结算
- `collateral_vault` 模块：管理发行人抵押品，确保偿付能力
- 关键数据结构：`BondContract { face_value: Coin<T>, coupon_rate: u64, maturity: u64, payment_frequency: u64, issuer: address, collateral: Coin<T> }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK
- 关键页面：Bond Marketplace（债券市场）、Issue Bond（发行债券）、My Bonds（我的债券）、Payment Calendar（付息日历）
- 后端 Keeper 服务在付息日和到期日触发合约操作

## 技术架构

发行人通过 PTB 创建债券合约，存入抵押品并设置债券参数（面值、利率、到期日）。合约铸造债券代币供投资者购买，代币持有者即为债权人。每个付息日，Keeper 通过 PTB 触发付息模块，合约自动从发行人抵押品中提取利息分配给持有人。到期日，合约自动赎回本金并销毁债券代币。如果发行人抵押不足，进入清算流程保护持有人。所有操作通过链上事件通知。
