# 链上薪资系统 / On-Chain Payroll System

> 企业发工资到链上，员工可选择收币种（SUI/USDC/BTC），自动兑换和分发

## 解决什么问题

加密货币企业和 DAO 的薪资发放面临多币种兑换和批量转账效率问题。财务人员需要手动为每个员工计算金额、执行兑换和逐笔转账，耗时且容易出错。跨境员工还面临汇率损失和到账延迟。链上薪资系统让企业一次操作即可完成全员工资发放，支持多币种自动兑换，员工可自主选择收款币种。

## 核心功能

- 批量薪资发放：通过 PTB 一次性向数十名员工发放工资
- 自动币种兑换：根据员工偏好自动通过 DeepBook 兑换为目标币种
- 薪资流集成：支持将月薪转为按日/按周的流式支付
- 税务自动计算：根据员工所在地区自动预扣税率并记录

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| PTB | 原子化执行"批量分发 → 逐个兑换 → 转账到员工地址" |
| Coin 标准 | 统一处理多种薪资代币 |
| 对象所有权 | PayrollConfig 作为企业对象，存储员工列表和配置 |
| 动态字段 | 存储员工偏好（目标币种、地址、税率） |
| 共享对象 | 企业薪资池作为共享对象 |

## 实现方案

### 智能合约 (Move)
- `payroll_factory` 模块：企业注册和薪资计划配置
- `employee_registry` 模块：管理员工列表、币种偏好和薪资等级
- `batch_disbursement` 模块：批量发放逻辑，支持多员工并行
- `auto_convert` 模块：通过 DEX 集成实现自动币种兑换
- 关键数据结构：`PayrollBatch { company: address, employees: vector<EmployeePayment>, total_amount: Coin<T>, executed_at: u64 }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK
- 关键页面：Company Dashboard（企业管理台）、Employee List（员工管理）、Run Payroll（执行发薪）、Employee Portal（员工自助门户）
- 后端提供 CSV 导入、税率计算和发薪提醒服务

## 技术架构

企业管理员在后台添加员工信息（地址、薪资、目标币种），设置发薪周期。发薪日管理员点击执行，系统构建 PTB：从企业资金池中为每位员工分配对应金额 → 对于需要兑换的员工，调用 DEX 将原始币种兑换为目标币种 → 将最终金额转入员工地址。整个批处理原子化完成。员工可在自助门户查看工资明细、选择币种偏好和查看历史发放记录。
