# 风险平价投资组合 / Risk Parity Portfolio

> 根据用户风险偏好，自动分配资产到不同 DeFi 策略，定期再平衡

## 解决什么问题

DeFi 用户面临众多投资选择但缺乏科学的资产配置方法。多数人要么全仓单一资产，要么随意分散，无法根据自己的风险承受能力做出最优配置。传统风险平价策略依赖专业基金经理，门槛高且收费昂贵。风险平价投资组合将现代投资组合理论编码到 Move 合约中，根据用户风险偏好自动分配和再平衡。

## 核心功能

- 风险评估问卷：用户回答几个问题确定风险偏好等级（保守/稳健/激进）
- 自动资产配置：根据风险等级分配到不同类型的 DeFi 策略（稳定币/蓝筹/高收益）
- 定期再平衡：资产偏离目标比例时自动调整回最优配置
- 组合业绩追踪：展示组合净值曲线、Sharpe 比率、最大回撤等指标

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | Portfolio 作为独立 Object，存储用户配置和持仓 |
| PTB | 原子化执行"卖出偏离资产 → 买入目标资产 → 更新持仓" |
| Coin 标准 | 支持多种资产类型的管理 |
| 动态字段 | 存储风险偏好、目标配置比例和再平衡历史 |
| 共享对象 | 策略注册池作为共享对象 |

## 实现方案

### 智能合约 (Move)
- `portfolio_factory` 模块：根据风险偏好创建投资组合
- `risk_engine` 模块：计算资产波动率和相关性，确定最优配置
- `rebalancer` 模块：检测偏离并执行再平衡交易
- `performance` 模块：计算净值、收益率和风险指标
- 关键数据结构：`Portfolio { owner: address, risk_level: u8, holdings: vector<Position>, target_allocation: vector<Allocation>, last_rebalance: u64 }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Recharts + D3.js
- 关键页面：Risk Assessment（风险评估）、Portfolio Overview（组合总览）、Rebalance History（再平衡记录）、Performance Analytics（业绩分析）
- 后端提供波动率计算和再平衡触发服务

## 技术架构

用户完成风险问卷后，risk_engine 根据答案确定风险等级和目标配置比例。用户存入资金，合约按目标比例分配到各 DeFi 策略。后端持续监控各资产权重，当偏离超过阈值（如 5%）时触发 PTB 再平衡：卖出超配资产 → 买入低配资产。每次再平衡记录在链上动态字段中。前端展示组合净值曲线和各项风险指标，帮助用户理解投资表现。
