# 杠杆收益策略 / Leveraged Yield Strategy

> 通过借贷协议放大收益，AI 控制杠杆率，自动降杠杆防清算

## 解决什么问题

成熟的 DeFi 用户希望获得高于常规策略的收益率，但手动管理杠杆风险需要 24/7 监控和快速反应。杠杆操作稍有不当就可能被清算，损失全部抵押品。杠杆收益策略通过 AI 驱动的杠杆管理，自动在最优杠杆率运行，在市场波动时快速降杠杆，在行情稳定时适度加杠杆，最大化收益的同时将清算风险控制在最低。

## 核心功能

- AI 杠杆决策：根据市场波动率、流动性和历史数据动态计算最优杠杆率
- 自动杠杆调整：市场波动增大时自动降杠杆，波动减小时适度加杠杆
- 多策略组合：支持杠杆借贷挖矿、杠杆流动性提供等多种策略
- 清算保护：多级安全网确保在任何极端行情下都不会被清算

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | LeveragedPosition 作为独立 Object，管理杠杆仓位 |
| PTB | 原子化执行"借款 → 投资 → 收益收割 → 还款"全流程 |
| Coin 标准 | 处理抵押品、借款和收益代币 |
| 预言机集成 | 通过 Pyth 获取实时价格计算杠杆率和健康因子 |
| 动态字段 | 存储杠杆参数、AI 决策历史和风险指标 |

## 实现方案

### 智能合约 (Move)
- `leverage_manager` 模块：创建和管理杠杆仓位
- `risk_controller` 模块：计算健康因子，执行杠杆调整
- `strategy_executor` 模块：执行具体的杠杆收益策略
- `emergency_module` 模块：极端行情下的紧急降杠杆和平仓
- 关键数据结构：`LeveragedPosition { collateral: Coin<T>, borrowed: Coin<T>, leverage_ratio: u64, strategy_type: u8, health_factor: u64 }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK + Python（AI 模型）
- 关键页面：Leverage Dashboard（杠杆总览）、Strategy Selection（策略选择）、Risk Monitor（风险监控）、AI Decisions（AI 决策日志）
- 后端 AI 模型计算最优杠杆率，Keeper 执行调整

## 技术架构

用户存入抵押品后选择策略和风险偏好，合约通过借贷协议借入更多资金放大投资。AI 模型持续分析市场数据（波动率、流动性、相关性），通过 Keeper 将杠杆调整指令提交到合约。合约通过 PTB 原子化执行调整：降低杠杆时部分卖出资产还款；提高杠杆时增加借款投资。多级安全网确保健康因子始终高于安全线。前端实时展示杠杆率、健康因子和 AI 决策理由。
