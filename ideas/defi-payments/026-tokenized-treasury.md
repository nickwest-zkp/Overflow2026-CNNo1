# 代币化国库券 / Tokenized Treasury Bills

> 将链下国库券收益映射到链上，用户通过 Sui 对象持有和交易

## 解决什么问题

国库券等传统固定收益产品收益率相对稳定且风险低，但链上用户难以直接投资传统金融市场。传统国库券购买需要银行账户、KYC 认证，且结算周期长。代币化国库券将链下国库券的收益权映射到 Sui 链上，用户通过持有链上代币即可享受国库券收益，无需银行账户和 KYC，且可在链上自由交易。

## 核心功能

- 国库券代币化：将链下国库券收益包装为链上代币，1:1 锚定面值
- 收益自动分配：国库券利息按持有比例自动分配给代币持有者
- 链上交易：代币可在 Sui DEX 上自由买卖，流动性由 AMM 提供
- 到期赎回：国库券到期时代币自动按面值赎回为稳定币

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | TBillToken 作为 Coin 对象，代表国库券份额 |
| PTB | 原子化执行"购买 → 铸造代币 → 分配收益" |
| Coin 标准 | 国库券代币使用 Coin<T>，可在 DEX 交易 |
| 动态字段 | 存储国库券元数据（到期日、票面利率、ISIN） |
| 时间锁 | 基于时间戳控制利息分配和到期赎回 |

## 实现方案

### 智能合约 (Move)
- `tbill_factory` 模块：创建新的国库券代币系列
- `mint_redeem` 模块：处理代币铸造（购买）和赎回（到期）
- `yield_distributor` 模块：按持有比例分配利息收益
- `reserve_manager` 模块：管理链下资产对应的链上储备证明
- 关键数据结构：`TBillToken { face_value: u64, coupon_rate: u64, maturity: u64, isin: String, reserve_proof: vector<u8> }`

### 前端 / 后端
- 技术栈：Next.js + @mysten/dapp-kit + Sui TypeScript SDK
- 关键页面：TBill Market（国库券市场）、My Holdings（我的持仓）、Yield Dashboard（收益总览）、Redemption（赎回管理）
- 后端管理链下储备证明和利息支付

## 技术架构

合规机构在链下购买国库券后，通过 reserve_manager 在链上创建对应的代币系列，存入储备证明。用户通过 PTB 使用稳定币购买国库券代币，合约铸造等额代币给用户。每个付息日，后端触发 yield_distributor 将利息按持有比例通过 PTB 分配给所有持有者。到期日，用户可通过 PTB 销毁代币按面值赎回稳定币。代币可在 DEX 上自由交易，价格由市场供需决定。
