# Investment Style Learner / 投资风格学习器

> Agent 学习用户的交易习惯和风险偏好，通过 Walrus 积累投资画像，推荐越来越贴合用户风格

## 解决什么问题

通用投资建议无法匹配每个用户独特的风险承受能力和投资风格。新手投资者不清楚自己的风格，老手则需要个性化的策略建议。本方案通过 AI Agent 持续学习用户的链上交易行为和对话偏好，构建动态投资画像存储在 Walrus，使推荐越来越精准。

## 核心功能

- 链上行为分析：自动分析用户的链上交易历史，提取交易风格特征
- 风险偏好评估：基于持仓集中度、交易频率、止损行为等维度评估风险偏好
- 风格画像构建：生成包含投资类型、偏好赛道、交易频率的多维度画像
- 个性化推荐：基于画像推荐匹配的投资策略和项目

## 使用的 Walrus / Sui 能力

| 能力 | 用途 |
|------|------|
| Walrus Blob 存储 | 存储用户投资画像、行为分析报告、推荐历史 |
| Sui 链上数据 | 读取用户地址的交易历史和持仓信息 |
| Seal 加密 | 加密投资画像等敏感金融数据 |
| MemWal 记忆层 | 缓存用户近期偏好变化，加速画像更新 |

## 实现方案

### Agent / AI 部分
- 链上分析 Agent 通过 Sui RPC 获取用户地址的完整交易历史
- 行为建模 Agent 使用聚类算法将用户行为归类（保守/激进/投机/长期持有）
- 风险评估 Agent 计算用户的风险偏好评分（基于最大回撤、持仓分散度等）
- 推荐生成 Agent 基于画像和当前市场状况生成个性化投资建议
- 使用 LLM 将分析结果转化为自然语言的投资风格描述

### Walrus 集成
- 投资画像以 `profiles/{user_id}/profile.json` 存储
- 行为分析以 `profiles/{user_id}/analysis/{date}.json` 存储
- 推荐历史以 `profiles/{user_id}/recommendations/{date}.json` 存储
- 风险评估以 `profiles/{user_id}/risk_score.json` 存储

### 前端 / 后端
- 前端：React + Recharts 雷达图，展示多维度投资画像
- 后端：Python FastAPI + scikit-learn 行为建模
- 关键页面：投资画像页、风格分析页、推荐列表页、历史追踪页

## 技术架构

用户连接钱包地址 → 链上分析 Agent 获取交易历史 → 行为建模 Agent 提取风格特征 → 风险评估 Agent 计算风险偏好 → 画像数据使用 Seal 加密 → 加密画像存入 Walrus → 推荐生成 Agent 基于画像生成建议 → 建议历史存入 Walrus → 前端展示画像和推荐。
