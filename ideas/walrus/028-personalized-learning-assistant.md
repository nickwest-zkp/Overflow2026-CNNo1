# Personalized Learning Assistant / 个性化学习助手

> 记住用户已学内容和进度，下次对话从断点继续，完整学习记录存储在 Walrus

## 解决什么问题

在线学习缺乏持续性和个性化，学习者经常因为中断后忘记学到哪里而放弃。传统 AI 助手无法记住上次的学习进度，每次都要从头开始。本方案将用户的完整学习轨迹持久化在 Walrus，AI Agent 自动跟踪学习进度，在任何设备上都能无缝继续学习。

## 核心功能

- 学习进度追踪：自动记录用户学过的章节、完成的练习、掌握程度
- 断点续学：新会话自动从上次停止的地方继续，Agent 简要回顾关键内容
- 自适应内容：根据掌握程度调整内容难度，弱项自动增加练习
- 学习路径规划：AI 根据学习目标生成个性化学习路径，动态调整

## 使用的 Walrus / Sui 能力

| 能力 | 用途 |
|------|------|
| Walrus Blob 存储 | 存储学习进度、课程内容、练习记录 |
| Sui 对象模型 | 链上记录学习成就、证书、学习 NFT 徽章 |
| MemWal 记忆层 | 管理短期学习上下文和知识点关联 |
| Seal 加密 | 保护用户学习数据隐私 |

## 实现方案

### Agent / AI 部分
- 教学内容 Agent 根据学习目标生成课程大纲和具体章节内容
- 评估 Agent 通过问答和练习测试用户掌握程度
- 推荐 Agent 根据弱项推荐补充材料和额外练习
- 路径规划 Agent 使用知识图谱技术规划最优学习路径
- 回顾 Agent 在新会话开始时生成简洁的前次内容回顾

### Walrus 集成
- 学习进度以 `learning/{user_id}/progress/{course_id}.json` 存储
- 课程内容以 `courses/{course_id}/chapters/{N}.md` 存储
- 练习记录以 `learning/{user_id}/exercises/{course_id}.json` 存储
- 学习路径以 `learning/{user_id}/paths/{goal_id}.json` 存储
- 成就数据以 Sui NFT 形式铸造（完成课程获得徽章）

### 前端 / 后端
- 前端：React + 交互式练习组件，支持 Markdown 内容渲染
- 后端：Python FastAPI，提供学习进度管理和内容生成 API
- 关键页面：学习仪表盘、课程浏览页、学习进度页、成就展示页

## 技术架构

用户设定学习目标 → 路径规划 Agent 生成学习路径 → 路径存入 Walrus → 教学内容 Agent 按章节生成内容 → 内容存入 Walrus → 用户学习并完成练习 → 评估 Agent 测试掌握度 → 进度更新存入 Walrus → 下次会话时回顾 Agent 读取进度 → 从断点继续教学。
